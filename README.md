## 介绍
  如果你的Chrome装了太多扩展，但是日常用到也就其中一部分，而这里面又有一部分是使用频率很低的，不适合一直开启，但是禁用之后，需要用到的时候找起来太麻烦并且用完再禁用这一系列操作太繁琐，那么这个扩展可能就适合你了，先把你那些日常需要用到的但是使用频率又比较低的扩展添加到收纳盒里，便可随时快速开启/禁用，非常方便。
